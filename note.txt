一、TypeScript 特性
(一)Alternative to javaScript(superset).
(二)Allow us to use strict types. // 嚴格規定型別
(三)Supports modern features (arrow functions, let, const) // 支援新語法
(四)Extra features(generics, interface, tuples etc)

二、先備知識
(一)Arrow functions // ES6箭頭函式
(二)The Dom // Document Object Model
(三)classes

三、前置準備
(一)安裝 node.js
(二)安裝 TypeScript
$ npm install -g typescript

四、相關指令與設定
(一)ts檔轉為js檔指令
$ tsc 檔名.ts 檔名.js 
// tsc 檔名.ts 也可work，會自動產生同檔名js

檔轉時若出現".ps1 檔案無法載入，因為這個系統上已停用指令碼執行。"問題
1.開啟PowerShell ISE，輸入指令get-executionpolicy，應該會顯示Restricted。
2.輸入指令set-executionpolicy remotesigned，更改執行原則即可。

(二)存檔時，ts檔自動轉為js檔指令
$ tsc 檔名.ts -w 

(三)存檔時，指定資料夾中所有ts檔自動轉為js檔指令
1.產生tsconfig.json檔(設定參數用)
$ tsc --init

2.設定ts檔存取位置及js檔產生位置(於tsconfig.json檔設定)
"outDir": "./public",
"rootDir": "./src",

(三)完成前一步驟，輸入以下指令即可在存檔時將src資料夾所有ts檔，轉為js檔儲存於public資料夾
$ tsc -w

(四)限制在src資料夾中的ts檔才會轉成js檔(於tsconfig.json檔設定)，須加上
"include": ["src"]
